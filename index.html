Guia de Configuração

Passos para configurar:

  1. Acesse Google Cloud Console
  2. Crie um novo projeto chamado "IronFace"
  3. Ative a "Google Drive API"
  4. Configure a tela de consentimento OAuth
  5. Crie credenciais OAuth 2.0 (Client ID)
  6. Crie uma API Key
  7. Adicione as URIs autorizadas
  8. Copie as credenciais no card ao lado

Banco de Dados

Como funciona:

O banco de dados é armazenado no seu Google Drive pessoal na pasta IronFace_Database.

  • Cada cadastro é uma imagem com metadados
  • Organização automática por cidade/ano/mês
  • Cidade é campo obrigatório para organização
  • Múltiplas abordagens geram múltiplas imagens
  • Busca por CPF, Nome e outros filtros
  • Acesso direto pelo Google Drive
  • Backup automático do Google

Sistema de Cache:

O sistema armazena descritores faciais processados localmente para acelerar carregamentos futuros.

  • Primeira carga: processa todas as imagens
  • Próximas cargas: até 90% mais rápido
  • Cache automático por imagem
  • Atualiza quando imagem é modificada

Importante:

  • Cada usuário tem seu próprio Google Drive
  • Os dados NÃO são compartilhados entre usuários
  • Token de acesso expira após 1 hora
  • Recomendado: usar conta Google institucional
